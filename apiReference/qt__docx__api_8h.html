En este archivo se incluyen las declaraciones de las clases correspondeintes a la API QT DOCX Esta API brinda soporte a las funcionalidades:

  • Pagina con tamaño y margenes
  • Encabezado y pie de pagina
  • Texto con formato
  • Imagenes
  • Listas
  • Tablas
